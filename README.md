# 🏋️ SFitNex — Your Intelligent Fitness Companion

🌐 **Live Demo:**  
https://smart-fit-ai-kappa.vercel.app/

SFitNex is an AI-powered fitness assistant designed to help:
- 💪 fitness enthusiasts
- 🏃 beginners starting their workout journey
- 🎯 users seeking personalized guidance

The platform uses gesture recognition, computer vision, and intelligent recommendations to make fitness training interactive, accessible, and effective.


## 📌 Problem
Many people struggle to:
- maintain proper exercise form
- create effective workout routines
- stay motivated without a trainer
- find nearby gyms and fitness resources


## 🎯 Objectives
✔️ Provide real-time exercise guidance  
✔️ Enable hands-free interaction through gestures  
✔️ Offer personalized workout recommendations  
✔️ Make fitness accessible to everyone  



## ✨ Core Features

### 🎥 AI-Powered Exercise Guidance
- Real-time pose detection
- Form correction feedback
- Step-by-step workout instructions

### 👋 Gesture Recognition
- Hands-free control
- Intuitive navigation
- Seamless workout experience

### 📊 Personalized Training
- Custom workout plans
- Progress tracking
- Adaptive recommendations based on:
  - ✔️ Fitness level
  - ✔️ Goals
  - ✔️ Available equipment

### 🍎 Nutrition Support
- Diet recommendations
- Calorie tracking tips
- Healthy meal suggestions

### 📍 Gym Finder
- Discover nearby fitness centers
- View ratings and reviews
- Get directions

## 🔄 System Flow

```
User → starts workout session
        ⬇️
Camera captures movements
        ⬇️
AI analyzes pose & form
        ⬇️
Provides real-time feedback
        ⬇️
Tracks progress & adapts
```

## 🚀 Getting Started

1️⃣ **Clone the repository**
```bash
git clone https://github.com/premkumar777-sys/smartfit-hub.git
cd smartfit-hub
```

2️⃣ **Install dependencies**
```bash
pip install -r requirements.txt
```

3️⃣ **Run the application**
```bash
python app.py
```

4️⃣ **Open in browser**
```
http://localhost:5000
```
