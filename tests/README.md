# 🧪 Testing Suite

This directory contains testing suites and instructions for SmartFit AI.

## Automated Tests

We use Vitest and React Testing Library for frontend component and integration tests.

### Running Tests

To launch unit and integration tests (when configured):
```bash
npm run test
```

## Manual Verification Flows

Refer to individual test plans in `/docs` for validation guidelines of interactive features (e.g. computer vision pose models).
